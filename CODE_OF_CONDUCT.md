# 贡献者公约行为准则

## 我们的承诺

为了营造开放、友好的环境，我们作为贡献者和维护者承诺：
无论年龄、体型、残障、种族、民族、性别特征、性别认同与表达、经验水平、教育背景、社会经济状况、国籍、外貌、宗教或性取向如何，所有人在项目与社区中的参与都应免受骚扰。

## 我们的行为标准

有助于构建积极社区氛围的行为包括：

- 使用友好、包容的语言
- 尊重不同观点与经历
- 坦然接受建设性批评
- 关注对社区最有益的事情
- 对其他社区成员保持同理心

不可接受的行为包括：

- 使用带有性暗示的语言或图像，以及不受欢迎的性关注或性挑逗
- 挑衅、人身攻击、侮辱性/贬损性评论，或政治攻击
- 公开或私下骚扰
- 未经明确许可公开他人的隐私信息（如住址、电子邮箱等）
- 在专业环境下可被合理认定为不当的其他行为

## 维护者职责

项目维护者有责任明确可接受行为的标准，并应针对不可接受行为采取适当且公平的纠正措施。

项目维护者有权并有责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、Wiki 编辑、Issue 及其他贡献；也可对其认为不当、威胁性、冒犯性或有害的行为实施临时或永久禁令。

## 适用范围

本行为准则适用于项目空间内，以及个人代表项目或社区的公共场合。
代表项目或社区的情形包括但不限于：使用官方项目邮箱、通过官方社交媒体账号发言，或在线上/线下活动中以指定代表身份出现。
项目维护者可进一步定义和澄清“代表项目”的范围。

## 执行与反馈

若出现辱骂、骚扰或其他不可接受行为，请联系项目团队：burningokr@brockhaus-ag.de。
所有投诉都会被审查与调查，并根据具体情况给出必要且适当的响应。
项目团队有义务对举报人信息保密。
如有需要，具体执行政策可另行发布。

若项目维护者未能善意遵循或执行本准则，项目领导层其他成员可酌情采取临时或永久处理措施。
